关于伪类的说明

发表于
  • link 为访问的链接
  • visited 已访问的链接
  • hover 当鼠标移到连接上时
  • active 鼠标点击不松开时

用例：

<style>
a:link {
    color: red;
}

a:visited {
    color: blue;
}

a:hover {
    color: pink;
}

a:active {
    color: green;
}
</style>

结构伪类选择器：

  • first-child 第一个孩子
  • last-child 最后一个孩子
  • nth-child(n) 第n个孩子
  • nth-child(odd) 奇数的孩子节点

  • nth-child(even) 偶数的孩子节点

    <style>
li:first-child {  /*第一个孩子节点*/
    color: red;
}
li:last-child {  /*最后一个孩子节点*/
    color: blue;
}
li:nth-child(2) { /*第二个孩子节点*/
    color: green;
}
li:nth-child(n) { /*所有的孩子节点*/
    color: pink;
}
li:nth-child(odd)  {      /*奇数个孩子节点*/
    color: gray;
}
li:nth-child(even) {     /*偶数个孩子节点*/
    color: yellow;
}
</style>

  • text-indent 首行缩进

  • 1em 是一个字的宽度

    <style>
    text-indent: 2em;
</style>
  • 复用选择器

  1. 交集选择器 用.来确定

    用例：

    <style>
p.one {
color: red;
}
</style>
<p class="one">1</p>
<p>2</p>
<p>3</p>

  2. 并集选择器 用,来确定

    用例：

    <style>
    div,p,span {
        color: red;
    }
</style>

<div>1</div>
<p>2</p>
<span>3</span>

  3. 后代选择器 用空格来确定，可以选择子孙后代的节点

    用例：

    <style>
    div p {
        color: red;
    }
</style>

<div>
    <p>11</p>
</div>

  4. 子元素选择器 用>来决定，只选择儿子节点

    用例：

    <style>
    ul>li {
        color: red;
    }
</style>

<ul>
    <li>a</li>
    <li>a</li>
    <li>a</li>            
</ul>
  • 伪元素选择器 用::表示
  1. first-letter 首字母
  2. first-line 首行

  3. selection 选中

    <style>
    p::first-letter {
        color: red;
    }
    p::first-line {
        color: blue;
    }
    p::selection {
        color: pink;
    }

</style>

  • css的属性
    1.css层叠性 如果一个属性通过两个相同选择器设置到同一个元素上，那么这个属性就会选择距离结构更近的选择器

    <style>
    div {
        color: red;
    }
    div {
        color: blue;
    }
</style>

    这时div的颜色会选择距离结构更近的蓝色，而不是红色。

  1. css继承性 子标签会继承父标签的某些样式。text-、font-、line-以及颜色可以继承。

  2. css优先级 行内样式>id选择器>类选择器=伪类选择器>标签选择器

  • 文字阴影和盒子阴影的一些参数
  • 文字阴影<text-shadow>
    text-shadow: 水平距离 垂直距离 模糊距离 颜色

text-shadow: 10px 10px 2px rgba(0, 0, 0, 0.5)//设置半透明
  • 盒子阴影<box-shadow>

    box-shadow 水平距离 垂直距离 模糊距离 模糊尺寸 颜色 内/外阴影

    其中：
  1. 水平距离和垂直距离是必须要填的；
  2. 内/外阴影默认是外阴影，内阴影为inset
    box-shadow: 0 10px 15px rgba(0, 0, 0, 0,4) 3px;
  • position的属性值
  1. static 自动定位
    • 静态定位是网页中所有元素的默认定位方式，就是普通标准流的特性
  2. relative 相对定位。相对于原文档流的位置进行定位
    1. 相对定位是以自己的左上角为基点来移动(相对于自己的位置),也就是说相对定位的盒子任然在标准流中，它后面的盒子也还是以标准流的方式对待它。
    2. 相对定位可以通过边偏移来移动位置，但是其原来占有的位置继续占有
  3. absolute 绝对定位。相对于其上一个已经定位的父元素进行定位
    1. 若所有父元素都没有定位，以浏览器当前屏幕为准对齐。
    2. 如果父元素有定位，那么子元素根据最近的有定位的祖先元素进行定位
    3. 绝对定位可以通过边偏移移动位置，但是脱离原来位置，并且不再占有
    4. 让绝对定位的盒子水平居中的方法
      position: absolute;
  left: 50%;
  margin-left: -盒子宽度的一半px;
  4. fixed 固定定位。相对于浏览器窗口进行定位

近在学习了一段时间的bootstrap之后发现，JavaScript这个对我还是非常的陌生，感觉自己有点学不下去了，所以我还是决定把JavaScript仔细的研究一下然后再去研究其他的框架和结构。
  然后我本来想去研读《你不知道的JavaScript》这本书的，但是看了开头发现并不适合初学者去学习，所以我的学习方式还是以慕课+实践这两部分来完成。

JavaScript的入门级教程

1、入门须知

  • 怎么插入js代码
    1. 写在 scrit 标签中
    2. 通过创建单独的javascript文件然后将代码引入html文件中
  • 放置的位置
    理论上可以将代码放在文件的任意位置，但是为了规范操作、于浏览器解析一般选择将引入的位置放在 head中

  • 打开浏览器窗口

    语法：windows.open(url,窗口名称，参数字符串)

    参数解释：

    1. url 在窗口中要显示网页的网址或路径。若果是空字符串，那么窗口不显示任何文档
    2. 窗口名称
    • 该名称有字母、数字、下划线组成
    • “_top”、”_blank”、”_self”具有特殊意义的名称，其中_top:在该网页的顶部显示目标网页；_blank:在新窗口显示目标网页；_self:在当前网页中显示目标网页
    • 相同的name的窗口只能创建一个，要想创建多个窗口则name不能相同
    • 参数表：

      window.open('thhp://www.fudashuai007.top','_blank','width=600,height=400,top=100,left=0,toolbar=no,menubar=no,status=no,scrollbars=yes')

  • 关闭窗口

    语法：window.close();

    • 关闭本窗口:window.close();
    • 关闭指定窗口：<窗口对象>.close();

      var x=window.open(‘http://www.fudashuai007.top');
      x.close;

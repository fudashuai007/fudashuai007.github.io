小白历险记之javascript《二》

  

在学习完javascript的一些基本的入门知识之后，感觉对javascript还是挺陌生的，特别是自己在看相关的代码的时候觉得自己离入门还早的很，所以这次还是想更深层次的学习一下javascript,希望能对加强自己能力有一定的帮助

事件

JavaScript 创建动态页面。事件是可以被 JavaScript 侦测到的行为。 网页中的每个元素都可以产生某些可以触发 JavaScript 函数或程序的事件。



主要事件表：



  • onclick()鼠标单击事件


    onclick是鼠标单击事件，当在网页上单击鼠标时，就会发生该事件。同时onclick事件调用的代码就会被执行，通常与按钮一起使用。

    <script type="text/javascript">
  function add2(){
    var numa,numb,sum;
    numa=6;
    numb=8;
    sum=numa+numb;
    document.write("两数和为:"+sum);  }
 </script>

 <input name="button" type="button" value="点击提交" onclick="add2()" />


  • onmouseover()&&onmouseout()<==>鼠标经过事件和鼠标移开事件


    标经过事件，当鼠标移到一个对象上时，该对象就触发onmouseover事件，并执行onmouseover事件调用的代码。
     鼠标移开事件，当鼠标移开当前对象时，执行onmouseout调用的代码。












    hit

  • onfocus()&&onblur()<==>光标聚焦事件和光标失焦事件


    当网页中的对象获得聚点时，执行onfocus调用的程序就会被执行。
     光标离开当前获得聚焦对象的时候，触发onblur事件，同时执行被调用的程序。











  • onselect()内容选中事件

    • 选中事件，当文本框或者文本域中的文字被选中时，触发onselect事件，同时调用的程序就会被执行。

    <script type="text/javascript">
function message(){
alert("您触发了选中事件！"); }
</script>

<textarea name="summary" cols="60" rows="5" onselect="message()">请写入个人简介，不少于200字！</textarea>


  • onchange()文本框内容改变事件

    • 通过改变文本框的内容来触发onchange事件，同时执行被调用的程序。

    <script type="text/javascript">
function message(){
alert("您改变了文本内容！"); }
</script> 

<textarea name="summary" cols="60" rows="5" onchange="message()">请写入个人简介，不少于200字！</textarea>

  • onload()&&onunload()<==>加载和卸载事件
    事件会在页面加载完成后，立即发生，同时执行被调用的程序。

    对象

    JavaScript 中的所有事物都是对象，如:字符串、数值、数组、函数等，每个对象带有属性和方法。

    对象的属性：反映该对象某些特定的性质的，如：字符串的长度、图像的长宽等；

    对象的方法：能够在对象上执行的动作。例如，表单的“提交”(Submit)，时间的“获取”(getYear)等；


    主要对象有：

    内置对象

    • Date日期对象

    • String字符串对象

    • Math对象

        • 浏览器对象

      • window对象

      • javascript计时器

          setTimeout()计时器

          clearTimeout()取消计时器

      • history对象

          • 返回前一个页面back()方法

          返回下一个浏览的页面forward()方法

          返回浏览器历史中的其他页面go()

      • location对象

      • Navigator对象

      • screen对象

          • 返回屏幕的高度screen.height

          返回屏幕的宽度screen.width

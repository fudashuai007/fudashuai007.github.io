关于javascript内置对象的理解还不是很透彻，所以专门拿出时间来学习一下有关的知识。

javascript内置对象

  • Date日期对象

    • 日期对象可以储存任意一个日期，并且可以精确到毫秒数


    访问方法语法: “<日期对象>.<方法>”
    常用方法：




  • 返回/设置年份get/setFullYear()

    var mydate=new Date();
    document.write(mydate.getFullYear()+”
    “);
    mydate.setFullYear(81);
    document.write(mydate+”
    “);
    注意
    1. 年份用四位数表示。
    2. 结果格式依次为：星期、月、日、年、时、分、秒、时区且不同浏览器，时间格式有差异。
  • 返回星期方法getDay()



    注意
    getDay()返回的是0-6的数字，0表示星期天

  • 返回/设置时间方法get/setTime()

    返回/设置时间，单位毫秒数，计算从 1970 年 1 月 1 日零时到日期对象所指的日期的毫秒数。


    单位为毫秒数
  • String字符串对象

  • 返回指定位置的字符

    stringObject.charAt(index)

    注意
    1. 字符串中第一个字符的下标是 0。最后一个字符的下标为字符串长度减一（string.length-1）。

    2. 如果参数index不在0与string.length-1之间,该方法将返回一个空字符串。
  • 返回指定的字符串首次出现的位置

    stringObject.indexOf(substring, startpos)
    说明
    1.该方法将从头到尾地检索字符串 stringObject，看它是否含有子串 substring。


    2.可选参数，从stringObject的startpos位置开始查找substring，如果没有此参数将从stringObject的开始位置查找。


    3.如果找到一个 substring，则返回 substring 的第一次出现的位置。stringObject 中的字符位置是从 0 开始的。
    注意
    1.indexOf() 方法区分大小写。


    2.如果要检索的字符串值没有出现，则该方法返回 -1。
  • 字符串分割split()

    stringObject.split(separator,limit)

    说明
    limit 表示分割的次数，spparator 表示从指定的地方分为字符串

    var mystr = “abc.defg.mn”;
    document.write(mystr.split(“.”)+”
    “);
    document.write(mystr.split(“.”, 2)+”
    “);
    结果:

    abc,defg,mn
    abc,defg

  • 提取字符串substring()

    用于提取字符串中介于两个指定下标之间的字符。



    stringObject.substring(start,stop)
    说明
    start 开始位置；stop结束位置，若没有设置，则一直到字符串结尾


    结果:

    stringObject.substr(start,length)
    说明
    start 要提取的子串的起始位置；length 要提取字符串的长度

一共有四种方法清除浮动

  • 额外标签法

    在浮动元素的下面添加一个空盒子，如<div style=”clear: both”>

    <style>
.father {
    width: 600px;
    background-color: pink;
}
.son1 {
    height: 200px;
    width: 200px;
    background-color: hotpink;
    float: left;
}
.son2 {
    height: 200px;
    width: 400px;
    background-color: blue;
    float: left;
}
.uncle {
    width: 600px;
    background-color: purple;
    height: 400px;
}
.clear {
    clear: both;
}


    <div class="father">
    <div class="son1"></div>
    <div class="son2"></div>
    <div class="clear"></div>
</div>
<div class="uncle"></div>

  • 给父元素添加overflow方法

    <style>
.father {
    width: 600px;
    background-color: pink;
    overflow: hidden;
}
.son1 {
    height: 200px;
    width: 200px;
    background-color: hotpink;
    float: left;
}
.son2 {
    height: 200px;
    width: 400px;
    background-color: blue;
    float: left;
}
.uncle {
    width: 600px;
    background-color: purple;
    height: 400px;
}
</style>

  • 使用after伪元素清除浮动

     .clearfix:after {
    content: "."; /*内容随便，尽量不要为空*/
    display: block;/*转化为块级元素*/
    height: 0;      /*不影响布局*/
    visibility: hidden;  /*内容隐藏*/
    clear: both;  /*清除浮动*/
}

  • 使用双伪元素清除浮动

    .clearfix:before, .clearfix:after {
    content: "";
    display: table; /*防止外边距合并*/
}
.clearfix:after {
    clear: both;
}

offset系列

未脱离文档流

offsetLeft:与父级元素的margin+父亲元素的padding+父亲元素的border+自己的margin有关，与自身的padding以及border宽度无关

#dv1 {
    width:200px;
    height:200px;
    backgroundcolor:#ccc;
}

#dv2 {
    width:100px;
    height:100px;
    backgroundcolor:#ddd;
}
<div id="dv1">
    <div id="dv2"></div>
</div>

dv2=document.getElementbyId("dv2");
  1. 当输出cosole.log(dv2.offsetLeft);
    输出结果应当为0
  2. 当给dv1一个”margin-left:100px;”,此时cosole.log(dv2.offsetLeft)
    输出结果为100,值得注意的是offset输出的结果是数值类型的，其后并没有任何单位
  3. 当给dv1一个”padding-left:100px;”,cosole.log(dv2.offsetLeft);
    输出的结果为200 父级元素(margin-left+padding-left)
  4. 当给dv1一个”border:20px solid red;”cosole.log(dv2.offsetLeft)
    输出的结果为220 父级元素(margin-left+padding-left+border)
  5. 当给dv2一个”margin-left: 30px;”cosole.log(dv2.offsetLeft)
    输出的结果为250 父级元素(margin-left+padding-left+border)+子级元素(margin-left)
  6. 当给dv2padding或者border任意的值时，最后得到的结果任然是250，子级元素的padding和border不影响offset的值
    脱离文档流
  • 子级元素脱离文档流主要是自己的margin和left的值

结构伪类选择器：

  • first-child 第一个孩子
  • last-child 最后一个孩子
  • nth-child(n) 第n个孩子
  • nth-child(odd) 奇数的孩子节点

  • nth-child(even) 偶数的孩子节点

    <style>
li:first-child {  /*第一个孩子节点*/
    color: red;

li:last-child {  /*最后一个孩子节点*/
    color: blue;

li:nth-child(2) { /*第二个孩子节点*/

li:nth-child(n) { /*所有的孩子节点*/
    color: pink;
}
li:nth-child(odd)  {      /*奇数个孩子节点*/
    color: gray;
}
li:nth-child(even) {     /*偶数个孩子节点*/
    color: yellow;
}
</style>

  • css的属性
    1.css层叠性 如果一个属性通过两个相同选择器设置到同一个元素上，那么这个属性就会选择距离结构更近的选择器

    <style>
    div {
        color: red;
    }
    div {
        color: blue;
    }
</style>

    这时div的颜色会选择距离结构更近的蓝色，而不是红色。

  1. css继承性 子标签会继承父标签的某些样式。text-、font-、line-以及颜色可以继承。

  2. css优先级 行内样式>id选择器>类选择器=伪类选择器>标签选择器

  • 文字阴影和盒子阴影的一些参数
  • 文字阴影<text-shadow>
    text-shadow: 水平距离 垂直距离 模糊距离 颜色

text-shadow: 10px 10px 2px rgba(0, 0, 0, 0.5)//设置半透明
  • 盒子阴影<box-shadow>

    box-shadow 水平距离 垂直距离 模糊距离 模糊尺寸 颜色 内/外阴影

    其中：
  1. 水平距离和垂直距离是必须要填的；
  2. 内/外阴影默认是外阴影，内阴影为inset
    box-shadow: 0 10px 15px rgba(0, 0, 0, 0,4) 3px;
  • position的属性值
  1. static 自动定位
    • 静态定位是网页中所有元素的默认定位方式，就是普通标准流的特性
  2. relative 相对定位。相对于原文档流的位置进行定位
    1. 相对定位是以自己的左上角为基点来移动(相对于自己的位置),也就是说相对定位的盒子任然在标准流中，它后面的盒子也还是以标准流的方式对待它。
    2. 相对定位可以通过边偏移来移动位置，但是其原来占有的位置继续占有
  3. absolute 绝对定位。相对于其上一个已经定位的父元素进行定位
    1. 若所有父元素都没有定位，以浏览器当前屏幕为准对齐。
    2. 如果父元素有定位，那么子元素根据最近的有定位的祖先元素进行定位
    3. 绝对定位可以通过边偏移移动位置，但是脱离原来位置，并且不再占有
    4. 让绝对定位的盒子水平居中的方法
      position: absolute;
  left: 50%;
  margin-left: -盒子宽度的一半px;
  4. fixed 固定定位。相对于浏览器窗口进行定位

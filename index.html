    小白历险记之css

    css边框

    css圆角边框

    主要语法为：border-radius

    <style>
div{
    text-aligin:center;
    border:2px solid #a1a1a1;
    padding 10px 40px;
    background:#dddddd;
    width:350px;
    border-radius:25px;
}
</style>

    css边框阴影

    主要语法为：box-shadow

    <style>
div{
    width:300px;
    height:100px;
    background-color:#ff9900;
    box-shadow:10px 10px 5 px #888888;
}
</style>

    css边框图片

    <style>
div{
    border:15px solid transparent;
    width:300px;
    padding:10 px 20 px;
}
#round{
    border-image:url()30 30 round;
    //border-image:url() 30 30 stretch;拉伸

}
</style>

    css背景

    • css background-size属性

      背景图片的尺寸

      background-size:100px 100px;//背景图片的大小
      background-size:40% 100%;//背景图片的拉伸

    • css background-origin属性

      定位区域

      background-origin:border-box;
      backgrroung-origin:padding-box;
      background-origin:content-box;

    • css多重背景图片

      background-image:url(path),url(path);

    css文本效果

    • css文本阴影

      h1{

      text-shadow:5px 5px 5px #ff0000;

      }

    • css自动换行

      p{

      word-wrap:break-word;
}

    css 2D转化

    角度转化
    主要函数为：rotate();
<style>
div{
    margin:30px;
    width:200px;
    height:100px;
    background-color:yellow;
    <b>transform:rotate(9deg);</b>旋转多少度
}
</style>
    坐标移动
    主要函数为translate();
<style>
div{
   width:100px;
   height:75px;
   background-color:yellow;
   border:1px solid black; 
}
div#div2{
    transform:translate(50px,100px);把元素从左侧移动50像素，从顶部移动100像素
}
</style>
    高度和宽度变换
    主要函数为skew();
<style>
div{
     width:100px;
   height:75px;
   background-color:yellow;
   border:1px solid black; 
}
div#div2{
    transform:skew(30deg,20deg);围绕X轴把元素翻转30度，围绕Y轴翻转20度
}
</style>
    旋转30度
    主要函数为matrix();
<style>
div{
     width:100px;
   height:75px;
   background-color:yellow;
   border:1px solid black; 
}
div#div2{
    transform:matrix(0.866,0.5,-0.5,0.866,0,0);将div元素旋转30度
}
</style>

    css 3D转换

    主要函数为rotateX()、rotateY();

<style>
div{
     width:100px;
   height:75px;
   background-color:yellow;
   border:1px solid black; 
}
div#div2{
    transform:rotateX(180deg);绕X轴旋转180度
}
</style>

<style>
div{
     width:100px;
   height:75px;
   background-color:yellow;
   border:1px solid black; 
}
div#div2{
    transform:rotateY(180deg);绕Y轴旋转180度
}
</style>

    css过渡

    主要内容为：transition：width Xs X表示过渡时间
<style>
div{
    width:100px;
    height:100px;
    background-color:yellow;
    transition:width 3s;
}
div:hover{
    width:300px;
}
</style>

    css动画

    背景色的改变
    两种颜色的改变：
<style>
div{
    width:100px;
    height:100px;
    background-color:yellow;
    animation:myfirst 5s;
}
@keyframes myfirst{
    from {background-color:yellow;}
    to {background-color:bule;}
}
</style>

多种颜色的改变：
<style>
div{
width:100px;
height:100px;
animation:myfirst 5s;
background-color:pink;
}
@keyframes myfirst{
    0% {background:pink;}
    50% {background:yelow;}
    100%{background:red;}
}
</style>
    位置的改变
    <style>
div{
    width:100px;
    height:100px;
    background-color:red;
    animation:myfirst 5s;
    position:relative;
}
@keyframes myfirst{
    0% {background-color:red;left:0px;top:0px;}
    25% {background-color:pink;left 200px;top:0px;}
    50%{background-color:green; left:200px;top:200px;}
    75% {background-color:gray;left:0px;top:200px;}
    100% {background-color:blue;left:0px;top:0px;}
}
</style>

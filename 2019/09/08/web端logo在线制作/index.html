Web端在线logo设计与制作系统

项目背景

logo是企业形象或者产品形象的符号化体现。logo最重要的一个功能就是识别性，便于消费群体或受众将不同企业的同类产品，或同一行业的不同企业区别开来。从而将logo所代表的产品或者企业打造成一。标志代表着企业的经营理念、文化特色、价值取向，反映企业的产业特点，经营思路，是企业精神的具体象征。因此一个好的logo能够代表一家公司良好的企业形象。而现如今logo的设计人工成本较大，制作时间偏长，还需要和设计师做充分的沟通与交流，过程复杂。而在互联网高速发展的今天，在线快速制作已经成为了可能，不仅可以实现快速的logo制作，而且成本低，时效快，而且制作过程方便，快捷。但目前市面上已有的logo设计制作还是存在着少许问题，因此需要一款更加便捷的logo设计系统。

项目目标

本项目的目标主要分为三大模块：
用户模块、设计者模块、管理模块
用户模块的主要功能是用户快速进行logo制作，主要包括logo布局设计、logo图案设计、logo文字设计。在本模块中，用户可以通过拖拽图案、选择文字的方式实现高速快捷的logo制作。在设计完logo初稿时可以预览实际的效果，最后可以选择保存或者尝试再次制作；
设计者模块的主要功能是实现设计者与买家用户的一个双方交易。设计者可以提供自己的logo设计图案，上传到本站点进行托管，当有买家想利用该设计者的设计图案时可以支付少量的金额。同时为了避免出现大量相同的logo图案，每位设计者的每件作品最多只能被引用3次；
管理模块是实现管理员是实现用户管理与系统管理的一个功能模块。

项目方案

预计采用的技术方案为vue+nodejs+elementUI

项目可行性

初步估计本项目的难度适中，目前认为难度较大的地方有：
1. 图案的拖动布局；
2. 根据用户的设定完成快速制作与推荐。用户进入系统可以选择快速布局，如上下结构、左右结构等；
3. 图形与文字的选择和变化。用户对于当前图形或者文字形式不满意时可以进行快速更换；
